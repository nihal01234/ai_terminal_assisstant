# ai_terminal_assisstant

AI terminal assisstant is a helper tool that can help you interact with a terminal in natural language, No need to memorize the codes just talk in the English language.
It also auto rectifies typos and meets the need.
Just run the code as administrator, have Ollama running in the background, open the commands.txt and run the command of need. This runs the commands in the windows terminal and gives you the outputs.
